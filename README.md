# Modulo-de-Banners-Magento
Modulo para upload de banners para magento 1.9

Modulo em desenvolvimento mas funciona perfeitamente!

Descrição: 
 - grupo de banners
 - titulo
 - url
 - data inicial
 - data final
 - descricao
 - cor
 - ordem
 - habilitado
 - upload de uma imagem de fundo
 - upload de uma imagem de 'produto'
